# 📚 Bookstore Inventory System

A C++ application that manages bookstore inventory using a custom Doubly Linked List implementation.

## Features

- Add books to inventory

- Remove books

- Search inventory

- Traverse inventory forwards and backwards

- Manage inventory records efficiently

## Technologies

- C++

- Object-Oriented Programming

- Doubly Linked Lists

- Dynamic Memory

- Data Structures & Algorithms

## What I Learned

This project strengthened my understanding of doubly linked lists by implementing insertion, deletion, and traversal operations while organizing a bookstore inventory system. It also reinforced object-oriented design and memory management concepts.
